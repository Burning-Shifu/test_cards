Задание

1. Разработать веб-интерфейс, на котором будут располагаться виджеты (карточки с минимальным набором данных), фильтры для виджетов, сортировка виджетов, а также формы для создания новых виджетов. Для хранения состояний и данных можно использовать LocalStorage. Дизайн страницы изображен на макете.

2. Поля в виджетах. Данные для карточек – это, условно, заказы, у которых имеются поля (ID, Дата создания, Номер накладной, Время прибытия на выгрузку, Тип заказа). Все поля (кроме ID и Даты создания) заполняются из формы создания. Типы заказа должны быть только: RUED, RUEX, RUSG.

3. Фильтры для виджетов. Виджеты должны фильтроваться по таким полям как: ID, Номер накладной, Время прибытия на выгрузку, Тип заказа.

4. Сортировка виджетов. Виджеты должны сортироваться по таким полям как: ID, Дата создания, Тип заказа. Положение должно сохраниться после перезагрузки страницы. Сортировка приоритетнее, чем перетаскивание вручную.

5. Перетаскивание виджетов. Виджеты можно перетаскивать и менять местами. Положение должно сохраниться после перезагрузки страницы.
